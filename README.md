# BorekV2
Borek Yazılımı HPGL, JPG, BMP ve EMF dosyalarını açmak için kullanılan bir yazılım


Yazılım bir görüntü aygıtı ile ilgili dosyaları gösterir. 
Kullanımı : İlk olarak dosyaların oluğu klasör seçilir.
Aç : İle okunacak dosya seçilir
